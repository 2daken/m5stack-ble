## About

Bluetooth low energy client for python.

## Requirements

* Python 3.5 or later.

## Install

```
$ git clone https://github.com/adafruit/Adafruit_Python_BluefruitLE
$ cd Adafruit_Python_BluefruitLE
$ sudo python setup.py install
```

## Usage

```
$ python main.py
```
